Welcome to aplanats's documentation!
====================================

NCBITax is just a lightweight version of the  ete3 library ncbitaxonomy
module and required dependencies.

Installation
------------

Aplanat is easily installed in the standard python tradition:

.. code-block:: bash

    git clone --recursive https://github.com/epi2me-labs/ncbitax.git
    cd ncbitax
    pip install -r requirements.txt
    python setup.py install
